if 基本概念介紹
  選擇型態分為:
  • 單向 if 敘述
  • 雙向 if - else 敘述           
  • 巢狀 if 敘述
  • switch 敘述
  • 條件運算式
*單向 if 敘述

 if ( boolean-expression ) {          
     statement(s);
 }

*流程圖
*雙向 if - else 敘述

 if ( boolean-expression ) {
     statement(s)-for-the-true-case; // 條件為true時
 }
 else {
     statement(s)-for-the-false-case; // 條件為false時
 }
